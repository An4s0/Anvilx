# Anvilx

Anvilx is an open-source monorepo project generator for scaffolding clean, scalable structures across multiple applications and services.

## Installation

```bash
npm install -g anvilx
```

## Usage

```bash
anvilx <command> [options]
```

### Commands

* `init <name>` : Create a new project
* `-h, --help`  : Show help message

### Examples

```bash
anvilx init my-app
anvilx -h
```
